# anvim

my neovim config :3 (nix is certified crazy)

try it out with

```sh
nix run git+https://git.koi.rip/koi/anvim
```

## Features

- up-to-date with latest neovim
- fast af (think so at least)
- full nix support
- [evergarden] :3
- LSP and stuff
- NO bloat
- much more...
